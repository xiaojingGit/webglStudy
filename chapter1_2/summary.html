一、学了哪些绘制

1、清空画布（我更愿意理解为用某种颜色初始化填充画布颜色）

2、绘制一个点（包含静态的点，通过鼠标点击绘制不同颜色的点）

二、用到的方法

1、gl.clearColor(1.0, 1.0, 1.0, 1.0)

2、gl.clear(gl.COLOR_BUFFER_BIT)

3、gl.getAttribLocation(gl.program, 'a_Position') // 获取变量a_Position的存储位置

4、gl.getUniformLocation(gl.program, 'u_FragColor')

5、gl.vertexAttrib[1234]f(a_Position, 1.0) // 1f:表示只传递一个值，代表x，2f:传递两个值，代表y。。。

6、gl.vertexAttrib[1234]fv(a_Position, [1.0, 0.0]) // fv:表示后面传递的值可以传递成数组，数组每项表示x,y,z,w

7、gl.uniform[1234]f(u_FragColor, 1.0)

8、gl.uniform[1234]fv(u_FragColor, [1.0, 1.0, 0.0, 1.0])

9、gl.drawArrays(gl.POINTS, 0, 1) // 绘制一个点
